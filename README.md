# CRUD operations REST django

### View movie list (read)
http://localhost:8000/movie/

### Add movie (create)
http://localhost:8000/movie/create

### Update movie (update)
http://localhost:8000/movie/update/<id>

### Get details of single movie (Read)
http://localhost:8000/movie/detail/<id>

### Delete Movie (delete)
http://localhost:8000/movie/delete/<id>
